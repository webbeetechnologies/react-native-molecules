## Getting Started

The monorepo has package-specific scripts. Follow the instructions in [packages/molecules/README.md](packages/molecules/README.md) for accurate install, dev, and test commands.

## Documentation

## Contributing

- Follow the lint rules enforced by `.github/workflows/lint.yml`.
- Use Node.js 20 and pnpm 9 for all development tasks.
- Open a draft PR early for feedback; mark it ready once lint/tests pass.

This project follows the [all-contributors](https://github.com/all-contributors/all-contributors) specification. Contributions of any kind welcome!
